Para rodar com Docker:

Abra o terminal na pasta raíz (arquivo SLN) e execute:

docker-compose -f docker-compose.prod.yml up --build --scale serverApi=4

(Este comando cria as imagens e roda os conteineres, uma vez que as imagens foram criadas e não existe a necessidade de atualizar nenhuma modificação, basta rodar o comando sem o --build

docker-compose -f docker-compose.prod.yml up --scale serverApi=4

Para rodar em modo DEV basta mudar a extensão do arquivo YML para dev
PS - É necessário criar os arquivos da pasta Node_Modules, para isto execute o NPM INSTALL na raiz da pasta CLIENT
